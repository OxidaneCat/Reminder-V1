# Reminder V1
This is a simple Reminder.

## Description
This is a reminder that allows you to add your upcoming events, and view what events you are curently having or upcoming in a simple list.
This reminder will automatically remove any events in the past.

## How to Run
Follow the instruction in the terminal by typing the options at the prompt and press Enter.

## Features
- Look at upcoming events in a simple list
- Add events
- Remove evemts
- View current date

## language
- English
